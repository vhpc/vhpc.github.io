Keynote Talk “What is RCU and How Does it Help the Linux Kernel Scale?", Paul E. McKenney, Software Engineer at Meta Abstract: RCU is used heavily as a replacement for reader-writer locking. But it is not a reader-writer lock. RCU is also used as a replacement for reference counting. But it is not a reference counter. RCU is used to provide type-safe memory. But it is not type-safe memory. RCU is used for many things, but it is none of those things.
